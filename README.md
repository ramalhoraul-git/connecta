# ✨ CONNECTA – Consultoria & Contabilidade Digital

> contadora formada pela **UFV – Campus Rio Paranaíba**.

## 🚀 Como Publicar

1. Faça upload dos arquivos `index.html`, `style.css` e `script.js`
2. Vá em **Settings → Pages**
3. Selecione branch `main` e pasta `/ (root)`
4. Clique em **Save**
5. Acesse: `https://seu-usuario.github.io/connecta-site/`

## ✅ Funcionalidades

- Design responsivo (Desktop, Tablet, Mobile)
- Carrossel de depoimentos com autoplay
- Carrossel de blog/atualidades
- FAQ com acordeão animado
- Formulário que envia direto pro WhatsApp
- Botão flutuante do WhatsApp
- Animações ao rolar a página
- Partículas no hero
- Máscara de telefone
- Menu hamburger mobile
- Botão voltar ao topo
